# 二分查找
这玩意哪怕一个没学过数据结构的计科生，在学习某些代码的时候，也会听过这个赫赫有名的算法，其实这个算法的核心已经写在了名字上，“**二分查找**”，也就是找到一个数组的中心点，判断 `target` 在中心点的右侧还是左侧，直到找到 `target`
## 所以要怎么使用二分查找呢 ？
也就是需要找到这个数组的长度，也就需要我们初始化两个指针，一个指向数组的头部 `i=0`, 还有一个指向数组的尾部 `j=n-1`。

> 这里我说的指针，其实不算是指针，可以是说是一个标记

## 又该如何判断呢 ？
1. 计算中点索引
2. 判断 `nums[m]` 和 `target` 的大小关系，分为以下三种情况。
    1. 当 `nums[m] < target` 时，说明 `target` 在区间  中，因此执行  。
    2. 当 `nums[m] > target` 时，说明 `target` 在区间  中，因此执行  。
    3. 当 `nums[m] = target` 时，说明找到 `target` ，因此返回索引  。
> 中心点的计算公式是 `m=[i+(j-i)/2]` 主要是因为防止大数越界

#需要我贴二分查找的代码吗 
```
#include <iostream>
#include <vector>
using namespace std;
int binarySearch(vector<int> &nums, int target)
{
    int i = 0;//初始化俩指针
    int j = nums.size() - 1;
    while (i <= j)//中止循环条件:i=j时
    {
        int m = i + (j - i) / 2;//计算终点(这个是关键哦)
        if (nums[m] < target)
        {
            i = m + 1;//缩小左边界
        }
        else if (nums[m] > target)
        {
            j = m - 1;//缩小右边界
        }
        else
        {
            return m;
        }
        return -1;
    }
}
```
